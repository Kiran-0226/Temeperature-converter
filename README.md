🌡️ Temperature Converter

A simple web-based Temperature Converter that allows users to convert temperatures between Celsius and Fahrenheit using HTML, CSS, and JavaScript.

📌 Features

Convert Celsius ➜ Fahrenheit

Convert Fahrenheit ➜ Celsius

Clean and simple user interface

Input validation for incorrect values

Real-time result display

Responsive centered layout

🛠️ Technologies Used

HTML5 – Structure of the webpage

CSS3 – Styling and layout

JavaScript (Vanilla JS) – Conversion logic and interactivity

⚙️ How It Works

Enter a temperature value in the input box.

Select the conversion type:

Celsius to Fahrenheit

Fahrenheit to Celsius

Click the Convert button.

The converted temperature will appear in the result box.

🔢 Conversion Formulas

Celsius ➜ Fahrenheit

F = (C × 9/5) + 32

Fahrenheit ➜ Celsius

C = (F − 32) × 5/9
